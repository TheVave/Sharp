Getting SharpPhysics

Full info on how to get SP

There are many ways to get SharpPhysics, but I'm going to be downloading a compiled release from the code.

After you get to the code, you can select Releases/tags and click on the most recent version.

After that click on {the name of the release}.zip, and it should automatically download the compiled code.

After you download the zip, you need to make sure you have Visual Studio.

If you do not have Visual Studio knowledge, then go to here.

And, one more, if you do not know C# I'd recommend starting from either the above link, or where I started (W3schools).

With that all out of the way, we can start on the window.

Creating a window (4 realz)

Creating a basic window is SP is very easy, it's just one line of code (not including the using statement).

The single line:

MainRendererSGL.InitRendering();

The line should create a window that looks something like this:


You have a console because, it is a console app.

There is a way to remove this. Right click on project->Properties and set output type to Windows Application.
