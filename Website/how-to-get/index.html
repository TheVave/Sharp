How to get SharpPhysics

* means is recommended

  1. From Github Code


    You can get SharpPhysics from Github directly, by going to main code and clicking the button, and then selecting download ZIP and extracting the code from there.

  2. From Github Release

    *

    You can get SharpPhysics from Github with Github releases. These are guaranteed to be able to compile, and have compiled and uncompiled source code.

  3. From Nuget

    *

    You can get SharpPhysics from nuget like any other package.
